# 今日菜单 · Daily Menu Card

一个 Claude Skill：把你随口说的一顿饭，变成一张像米其林品鉴菜单一样精致的可打印 HTML 菜单卡。

> "我今天吃了麦当劳的薯条和可乐" → 生成一张排版讲究、字体考究的"菜单"，认真到有点好笑，这种反差就是它的乐趣所在。

---

## 这是什么

这个 Skill 教 Claude 把任何一餐——无论是随便吃的快餐，还是认真做的家宴大菜——都用正式菜单卡的方式呈现出来：

- 自动把你说的菜分类成 **前菜 / 主菜 / 素菜 / 汤 / 甜品 / 饮品**（没有的类目自动跳过，不会硬凑）
- 每道菜下面配一行简短的食材说明（不是花哨的文案，是真实菜单那种克制的成分罗列）
- 饮品（如可乐、杏仁露）只写名字，不写食材描述
- 一次性问完语言、风格、菜单名三个问题，不会反复打断你
- 支持中文 / 英文 / 双语排版（双语时主语言大字在上，副语言斜体小字在下）
- 两种视觉风格可选：
  - **极简白**：纯白背景，居中排版，靠字体和留白撑起高级感
  - **暖调质感**：米杏色背景，金棕色点缀，更有家宴/年夜饭气氛
- 支持上传自己的背景图（直接叠加文字，或只提取图片配色来重新设计背景）
- 生成的是独立 HTML 文件，可保存、可打印、可分享

## 怎么用

把这个 Skill 放进你的 Claude Skills 目录后，直接跟 Claude 说今天吃了什么、或者准备做什么菜就行，比如：

```
我今天想吃龙虾、鱼子酱、海胆、和牛，你看着做四道菜，
甜品加个香草冰淇淋，给我做个菜单
```

或者一桌家宴：

```
今天做东北菜家宴：锅包肉、溜肉段、排骨豆角炖土豆、
小白菜汆丸子汤、东北大拉皮、气泡豆角，饮料喝杏仁露
```

Claude 会先确认一轮语言/风格/菜单名（一次问完，不会一项一项问），然后直接生成一张可下载的 HTML 菜单卡。

## 文件结构

```
daily-menu-card/
├── SKILL.md                      # Skill 的核心说明，Claude 读这个文件来执行
├── assets/
│   └── template.html             # 极简白风格的基础模板
└── references/
    └── style-presets.md          # 两种风格的完整设计规范（配色、字体、布局）
```

## 设计哲学

两种风格背后是同一句话：**克制才显高级**。真正的高端餐厅菜单往往惜字如金——不堆形容词，不堆装饰，让排版和留白自己说话。这个 Skill 刻意避免把每道菜写成小红书文案，而是用菜单该有的方式：菜名 + 一行食材，仅此而已。
